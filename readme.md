# Learn Git

## Working Directory

```.git```폴더를 제외한 나머지 공간.

작업 내용을 반영하고 싶으면 ```Staging Area```에 ```add```해야 한다.

```bash
git add 파일명
```


<br>

## Staging Area

```Working Directory```에서 ```add```된 변경 데이터에 대한 이력을 기록하기 위해 모아둔 공간.

```.git```폴더 내 ```index```에 기록된다.
